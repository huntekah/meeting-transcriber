# meeting-transcriber
## Version 0.3.0 - work in progress
Turn on during meeting. Get transcription live, and a text note later, to use with your note app.

An ASR meeting transcriber app.

![ASR demo](assets/asr_demo.gif)

## Features

- Listens to system audio and microphone
- Diarizes speakers
- Saves transcripts

We build robust, simple programs that won't break.
